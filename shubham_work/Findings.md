## `Findings` 

```
Question 3: Fast Food Restaurants Per Capita for All States
```

```
Objective :
```

```
The objective of this analysis was to determine which states have the
highest and lowest concentration of fast food restaurants relative to their
population.
```

## `Methodology :` 

```
Restaurant counts were calculated for each state using the fast food
restaurant dataset. State population data was merged with the restaurant counts.
The number of fast food restaurants per 100,000 residents was calculated using
the following formula:
```

```
Restaurants per 100,000 Residents = (Restaurant Count / Population) × 100,000
```

```
States were then ranked based on this metric.
```

```
Results :
```

```
The analysis revealed significant variation in fast food restaurant
density across states. Some states had a much higher concentration of
restaurants relative to their population, while others had lower restaurant
availability per capita.
```

```
The top-ranked states demonstrated a greater density of fast food restaurants,
indicating higher accessibility for residents. States with lower values had
fewer restaurants relative to their population size.
```

## `Conclusion :` 

```
Using a per-capita metric provides a more accurate comparison than simply
counting restaurants. Population-adjusted analysis highlights differences in
restaurant availability across states and provides a better understanding of
fast food distribution patterns.
```

Question 5: Major Cities with the Most and Least Fast Food Restaurants Per
Capita
```

```
Objective :
```

```
The objective of this analysis was to identify major cities with the
highest and lowest concentration of fast food restaurants relative to population
size.
```

## `Methodology :` 

```
Cities were grouped and restaurant counts were calculated. Only major
cities with populations greater than or equal to 500,000 residents were included
in the analysis.
```

```
The following formula was used:
```

```
Restaurants per 100,000 Residents = (Restaurant Count / Population) × 100,000
```

```
Cities were ranked based on the calculated restaurants-per-capita value.
```

## `Results :` 

```
The top 10 cities had the highest concentration of fast food restaurants
relative to their population. These cities provide greater access to fast food
establishments on a per-capita basis.
```

```
The bottom 10 cities had the lowest concentration of fast food restaurants among
major cities. Although some of these cities have large populations, the number
of restaurants relative to population size was comparatively lower.
```

```
Conclusion :
```

```
Per-capita analysis provides a more meaningful comparison than raw
restaurant counts because it accounts for population differences. The results
demonstrate that fast food restaurant density varies considerably across major
cities in the United States.
